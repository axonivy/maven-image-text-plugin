Content Object View

This view reacts to selection changes in the CMS Tree View and shows the values (in all languages) of the selected content object.

Accessibility

Window -> Show View -> Other -> Xpert.ivy -> CMS Object Details

Features

The Content Object View basically offers the same functionality as the Content Object Details part of the Content Editor (see there for a complete description of the available features).

Upon selection of a node in the CMS Tree View it shows the values of the selected content object in a list of collapsible sections together with some general information.

Warning

The view currently offers only limited editing support. It is therefore recommended to use the CMS Editor to modify content object values.
