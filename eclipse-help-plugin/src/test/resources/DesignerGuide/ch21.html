This chapter deals with configuration files of the Xpert.ivy Designer

Introduction

Configuration files are located in the configuration folder under the %XPERT_IVY_INSTALLATION% folder. These files you have to edit out of Xpert.ivy, e.g. using a system editor. It is recommended to stop your designer before you start changing these configurations.

There are some configuration files with extension *.xml. These are normal XML files. Meanwhile the *.any files are so called any files. There are a couple of any implementations - so that there is no real standard. The intension behind this is to have an easily to understand configuration with simple structure, and practically unlimited parameter types.

Tip

It is recommended to make a backup copy before you start changing configurations.
