Content Object Value Editors

The Content Object Details area of the CMS editor and the Content Object View will show different editors for values of the different Content Object Types. This section briefly introduces them.

String Editor

The string editor is simply a single-line text field; it does not accept line breaks. Content can not be imported.

Text Editor

The text editor has to views an Edit and a Source view. The Edit view is a WYSIWYG HTML text editor where you can format your text and the view represents the text appropriate. The Source view is a text only editor where you can edit the text low level. Both views are synchronized if you edit in the Edit view the Source View gets updated and vice versa. Contents cannot be imported.

Image Editor

The image editor shows images of the types GIF, PNG and JPG. Content import is supported. Images that are larger than the available editor area will not be resized and can not be scrolled. A clip of the image's top left corner will be displayed in original size in the available space.

Document Editor

Preview

The document editor is used for all document content objects (basically for binary data) as well as for the layout content type. The editor can show a preview of textual content (e.g. HTML , XML or JSP layout) and will also show information about the size and encoding of the displayed content. For binary document types (e.g. PDF, audio or video) a preview is not available.

Import from File

Importing of content is supported. The import will try to determine the encoding of the imported document by itself (for types that support it, e.g. HTML or XML). If the encoding cannot be determined, then the user is queried for it.

Tip

Change the file browser's view to show document miniatures to get previews of image files. This will greatly help you to select the image that you'd like to import (this feature is not supported on all platforms).

Export to file

Export of content is supported on the action of the editor's own export button (below the preview area). The export will try to determine the encoding of the exported document by itself (for types that support it, e.g. HTML or XML). If the encoding cannot be determined, then the user is queried for it.

Source Editor

The source editor is used to edit any kind of source text like jsp, html or jscript.

By clicking on the button the user can insert process data that can be selected from the opening attribute browser dialog. An optional condition may be specified as well as a suitable format for the type of the selected attribute (if available).

CSS Editor

The CSS editor is a simple text editor. You can import the content from a file.
