Xpert.ivy Project Deployment

To deploy an Xpert.ivy project to the Xpert.ivy Server execute the following steps:

  1. Export all files of the project you want to deploy to a zip file using the Export wizard of Xpert.ivy Designer (See next section).

  2. Copy the zip file with your project files to the Xpert.ivy Server.

  3. Start the Server Administrator application on the Xpert.ivy Server

  4. Choose or create an application

  5. Choose or create a process model

  6. Choose or create a process model version

  7. Open the detail page of the process model version and find the section Deployment.

  8. Press the Deploy button to start the deployment wizard.

  9. On the first step of the deployment wizard choose the zip file with your project files and follow the wizard to deploy your project.

Tip

More information about the deployment of Xpert.ivy projects or applications, process models and process model versions can be found in the Xpert.ivy Server Guide.
