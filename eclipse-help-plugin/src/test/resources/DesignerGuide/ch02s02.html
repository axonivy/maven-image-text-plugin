Xpert.ivy Archives

There are two different types of Xpert.ivy projects available. Normal Xpert.ivy projects are used to develop artifacts. Artifacts in those projects are changed frequently. Once the artifacts of a project are developed and stable you can export the normal Xpert.ivy project to a Xpert.ivy Archive. Xpert.ivy Archives are pre-built Xpert.ivy projects that are stored in one single *.iar file.

Xpert.ivy Archives can be imported to a Xpert.ivy workspace like normal Xpert.ivy projects. All artifacts of an Xpert.ivy Archive can be viewed but not edited. Xpert.ivy Archives already contain all built artifacts. Therefore, Xpert.ivy Archives do not have to be built or validated again in the workspace. Therefore, Xpert.ivy Archives will increase your workspace build, refresh and update time.

There are multiple ways to create or import Xpert.ivy Archives:

  • Xpert.ivy Archives can be exported and imported.

  • Xpert.ivy Projects can be packed (archived) or unpacked (unarchived) inside the workspace.
