Xpert.ivy Documentation

Now you are reading Xpert.ivy Designer documentation. In case you are want to know more on

  • Getting the latest Xpert.ivy Version: go to http://www.ivyteam.ch

  • System requirement: please read Readme.html in the installation directory

  • Working with Xpert.ivy Designer: Please get the Getting Started documentation at http://www.ivyteam.ch

  • Server administration: please read the ServerGuide (in installation folder of a server)

  • Upgrading an existing installation: please read Migration notes (located in installation folder).

All of above mentioned documentation is going to be brief and tend to describe only necessary functionality. We highly encourage reading these documentations to speed up your development, to get to know new features or to eliminate potential problems.

Tip

Do not forget, anytime you find some unknown features hitting F1 will show you a context sensitive compact help!
