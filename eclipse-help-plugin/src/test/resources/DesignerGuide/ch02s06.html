Exporting a Project

Overview

Xpert.ivy projects can be exported from the workspace to various output formats using the Export Wizard.

Accessibility

You can access the Export Wizard over the menu:

File -> Export ...

Features

For Xpert.ivy users the following output formats are useful:

General > Archive File

Exports projects to a *.zip or *.tar file.

General > File System

Exports projects to the file system.

Xpert.ivy > Xpert.ivy Archive (*.iar)

Exports a normal Xpert.ivy project to a Xpert.ivy Archive (*.iar file).
