CMS Tree View

This view shows the CMS of currently active project. The same information and manipulation actions are available in the tree of the left part of the CMS Editor.

Accessibility

Window -> Show View -> CMS

Features

The CMS tree view basically offers the same features as the tree of the CMS Editor (see there for a complete description of the available features).

However, the following features are additionally available on the CMS tree view only:

Content Object Values

The CMS tree view shows a leaf node for each value of a content object (i.e. English, German, French etc.). This way you can see immediately what language values that are defined for a content object.

The default value of a content object is marked with a small check mark () in the upper right corner.

Value Preview

For textual values (such as strings or texts) a short preview is displayed in an additional value column. This helps to identify the correct value, especially when searching.

Content Object Editor

Content objects can be edited by double clicking a content object. A content object editor will open and make content value manipulations available without opening the CMS Editor dialog.

Tip

Content objects (e.g. strings and images) from the CMS Tree View can be dragged and dropped onto the components of a Rich Dialog panel (e.g. Labels and Buttons). A reference to the dragged icon or text field will be automatically set on the target component, if applicable.
