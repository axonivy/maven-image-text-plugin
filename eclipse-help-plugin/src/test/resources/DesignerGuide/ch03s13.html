Reference View

Overview

The Reference view shows the references between the various Xpert.ivy project artifacts. A reference of an artifact is everything which is used/called from the artifact (e.g. call to a callable process or User Dialog) or which is embedded in the artifact (e.g. composite element in a process or processes inside a project). Inverse references are the opposite of references. This means an inverse reference of an artifact is everything which uses/calls the artifact or which contains it.

Features

The Reference view has the following functions:

Refresh ()

This function reloads the actual showed references.

Stop ()

This function stops the calculation of references.

Show References ()

This option shows the references of the actual root object.

Show Inverse References ()

This option shows the inverse references of the actual root object.
