Composite

The Composite element is located in the Activity drawer of the process editor palette.

Element Details

A composite element folds a part of a process into a box. It is also know as embedded sub process. This allows for hierarchical structuring of the process model into sub processes. Sub processes are obtained top down or bottom up. Either by selecting and combine part of a process or by inserting an (initially empty) composite element from the palette.
