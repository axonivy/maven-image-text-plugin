BPMN Task Elements

On the BPMN Tasks drawer are several BPMN Task elements available. They can be used to design a process at a high level, where details of the technical implementation are hidden inside the element itself. Thus BPMN Task elements behave similar to a Composite, but their purpose is different.

They are intensively used by the importer of Xpert.ivy Modeller Processes.

Available BPMN Task Elements

BPMN Task

A BPMN Task element can be used to indicate an undefined or unspecific BPMN task.

BPMN User Task

A BPMN User Task element can be used to indicate a step that implies execution by the user.

BPMN Manual Task

A BPMN Manual Task element can be used to indicate a step that implies execution without assistance of IT means.

BPMN Script Task

A BPMN Script Task element can be used to indicate a step that implies execution inside the process engine itself.

BPMN Receive Task

A BPMN Receive Task element can be used to indicate a step that implies reception of a message.

BPMN Business Rule Task

A BPMN Business Rule Task element can be used to indicate a step where a business rule is evaluated.

BPMN Send Task

A BPMN Send Task element can be used to indicate a step that implies sending of a message.

BPMN Service Task

A BPMN Service Task element can be used to indicate a step that implies calling an automated function.
